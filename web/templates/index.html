🌐 API REST Integrations

La API REST de CyberToolkit permite la integración con herramientas de seguridad externas (SOAR, SIEM, Burp Suite, CI/CD).

GET /api/tools
Retorna el catálogo completo de herramientas y su estado actual en formato JSON.
GET /api/reports
Lista todos los reportes de auditoría exportados (.json, .html, .txt).
